```python

# Proyecto: TryOnYou V10 OMEGA
# Patente: PCT/EP2025/067317
# Arquitecto: Rubén Espinar
# Estado: PRODUCTION_STABLE | Soberanía: tryonyou.pro
```

### Validación Automática

Se implementan **pre-commit hooks** para:

1. ✅ **Detección de headers faltantes** → Commit rechazado
2. ✅ **Escaneo de secretos** → Detección de credenciales hardcodeadas
3. ✅ **Linting de seguridad** → Flake8 + PyLint
4. ✅ **Validación de formatos** → Black, YAML, JSON

**Instalación de hooks:**

```bash
pip install pre-commit
pre-commit install
```
